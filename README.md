## О проекте

В этом репозитории собраны лабораторные работы по Flask (ЛР0–ЛР6).  
Каждая лабораторная находится в **отдельной папке** и представляет собой самостоятельное веб-приложение или этап развития проекта.

Основные темы:
- маршруты и шаблоны Jinja2
- работа с `request`
- формы и валидация
- аутентификация (`session`, `Flask-Login`)
- CRUD и роли пользователей (RBAC)
- журнал посещений и отчёты
- курсы, отзывы, пагинация, сортировка, миграции БД

## ЛР2

**Папка:** [`lab2/`](./lab2)

### Тема
Работа с объектом `request` во Flask + валидация номера телефона.

### Что реализовано
- просмотр параметров URL (`request.args`)
- просмотр заголовков запроса (`request.headers`)
- просмотр cookies (`request.cookies`)
- обработка данных формы (`request.form`)
- форма проверки номера телефона
- проверка допустимых символов
- проверка количества цифр
- форматирование номера в вид `8-***-***-**-**`
- вывод ошибок через Bootstrap (`is-invalid`, `invalid-feedback`)

### Основные файлы
- `app.py` — маршруты и логика обработки запросов/форм
- `templates/base.html` — базовый шаблон
- `templates/index.html` — главная страница
- `templates/inspect.html` — отображение request-данных
- `templates/form.html` / `templates/auth.html` — формы (если присутствуют)
- `static/shared.css` — стили

## Что означают основные файлы и папки

### Общие (почти во всех лабораторных)
- `app.py` — основной Python-файл Flask-приложения (маршруты, логика, запуск)
- `templates/` — HTML-шаблоны Jinja2
- `templates/base.html` — базовый шаблон (шапка, меню, footer)
- `static/` — CSS, JavaScript, изображения
- `*.db` — локальная база данных SQLite
- `requirements.txt` — список зависимостей Python (если используется)

### Для форм и шаблонов
- `templates/login.html` — форма входа
- `templates/user_form.html` — форма создания/редактирования записи пользователя
- `templates/user_view.html` — просмотр записи пользователя
- `templates/partials/_form_macros.html` — макросы Jinja2 для переиспользуемых полей формы
- `templates/visit_logs/*.html` — страницы журнала посещений и отчётов

### Для ролей и отчётов
- `rbac.py` — правила ролей и проверки доступа
- `reports.py` — генерация отчётов, агрегаций, CSV

### Для ЛР6 (модульный проект)
- `models.py` — описание моделей БД (таблиц)
- `courses.py` — маршруты и бизнес-логика по курсам/отзывам
- `auth.py` — аутентификация и вход
- `migrations/` — изменение схемы БД по версиям (Alembic)

### Служебные файлы/папки (обычно не нужны в Git)
- `__pycache__/` — кэш Python
- `.venv/`, `ve/` — виртуальные окружения Python
- `*.pyc` — скомпилированные файлы Python
- `*.bak*`, `*.save` — резервные/временные копии
- `*.log` — логи
- `gunicorn.ctl` — runtime-артефакт (сокет/служебный файл)
- `.env` — переменные окружения (секреты, пароли, ключи)
